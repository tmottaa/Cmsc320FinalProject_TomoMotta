2. Introduction¶

Topic: This tutorial explores the intersection of sports analytics and betting markets. We investigate whether an NBA player's recent momentum ("hot streaks") can predict whether they will score more points than their betting line.

Question: Can a machine learning model beat the Vegas "Over/Under" line using only simple recent form ideology.

Importance: If a simple model can predict this with 52.4% accuracy or greater (the break-even point for -110 odds), it implies that this model is effective. If not, it demonstrates that Vegas has the better model for sports betting.

5. Primary Analysis: Machine Learning¶

To answer our research question, we selected a Logistic Regression model.

Reasoning:

  • Binary Classification: Our target over_hit is binary (0 or 1).
  • Interpretability: Logistic Regression provides obvious impacts of the recent performance of nba players.
  • Baseline: It serves as a robust baseline to test if linear relationships exist between momentum and betting outcomes.

Features/Variables: We engineered features strictly from old data to avoid leakage:

  • pts_last_5: Rolling 5-game scoring average.
  • last_5_hit_rate: Rolling 5-game "Over" percentage (Momentum).
  • is_home: Home court advantage indicator.

final_df = pd.read_csv("nba_final_dataset.csv")

final_df["Date"] = pd.to_datetime(final_df["Date"], errors='coerce')
final_df = final_df.dropna(subset=["Date"])
final_df = final_df[final_df["PTS"] < 150]

final_df = final_df.sort_values(by=["Player", "Date"])

grouped = final_df.groupby("Player")
final_df["pts_last_5"] = grouped["PTS"].transform(lambda x: x.shift(1).rolling(5, min_periods=1).mean())
final_df["last_5_hit_rate"] = grouped["over_hit"].transform(lambda x: x.shift(1).rolling(5, min_periods=1).mean())

def check_away(row):
    return 0 if row.astype(str).str.contains("@").any() else 1

final_df["is_home"] = final_df.apply(check_away, axis=1)

final_df["pts_last_5"] = final_df["pts_last_5"].fillna(final_df["PTS"].mean())
final_df["last_5_hit_rate"] = final_df["last_5_hit_rate"].fillna(0.5)

features = ["is_home", "pts_last_5", "last_5_hit_rate"]
X = final_df[features]

7. Insights and Conclusions¶

Does the uninformed reader feel informed? Yes. We established that predicting NBA betting outcomes is harder than simply following trends.

  • Momentum Exists: The positive coefficient for last_5_hit_rate confirms that players on hot streaks are slightly more likely to go over and that momentum is real!
  • Market Efficiency: The fact that accuracy hovered near 50% proves that Vegas oddsmakers effectively considered momentumn and other variables, neutralizing our edge that we thought we had in this model.

Conclusion: While we successfully built a pipeline to scrape, clean, and model NBA data, we failed to beat the market and sportsbooks. This supports the hypothesis that public information is already included in betting lines and it will take a lot more complex and deeper analysis than just momentumn.
